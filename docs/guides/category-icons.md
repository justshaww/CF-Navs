# 分类与图标记录

`shaw的任意门` 当前使用的分类名称与 Iconify 图标代码。

## 分类图标

| 分类 | Iconify 图标 | 用途 |
| --- | --- | --- |
| 社区&论坛 | `flat-color-icons:conference-call` | NodeSeek、LinuxDo、V2EX 等社区论坛 |
| 服务器 | `flat-color-icons:data-configuration` | VPS 商家后台、服务器面板、Komari |
| 域名 | `flat-color-icons:globe` | 域名注册商、Cloudflare DNS |
| AI工具 | `fluent-color:bot-24` | ChatGPT、Claude、Gemini 等 AI 服务 |
| 中转站 | `fluent-emoji-flat:old-key` | API Key、接口中转和额度服务 |
| 秘密基地 | `flat-color-icons:home` | 使用 VPS 自建的站点 |
| 百宝袋 | `fluent-emoji-flat:toolbox` | IP、测速、DNS、临时邮箱等实用工具 |

## 备选图标

### 中转站

- 钥匙：`fluent-emoji-flat:old-key`
- 闪电：`noto:high-voltage`
- 加油站：`noto:fuel-pump`

### 百宝袋

- 工具箱：`fluent-emoji-flat:toolbox`
- 锤子与扳手：`fluent-emoji-flat:hammer-and-wrench`
- 扳手：`fluent-emoji-flat:wrench`

## 使用方法

1. 进入管理后台的“分类管理”。
2. 新增或编辑分类。
3. 将图标代码填入“Iconify 图标”输入框。
4. 点击“使用 Iconify”。
5. 点击“保存”。

这些图标通过 CF-Navs 的本地图标代理加载，不需要配置图床。
