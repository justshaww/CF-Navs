## 维护约定

- 页面与组件修改完成后先运行 `npm run type-check` 和 `npm run build`。
- 线上分类、书签和站点设置保存在 D1，重要调整前先导出备份。
- 登录会话保存在 KV；迁移站点时需要分别迁移 D1 数据和重新配置 KV。
- Cloudflare 生产部署使用 `main` 分支，部署后访问正式域名检查实际效果。
- `SETUP_TOKEN` 只用于首次安装或重新初始化，不要写入 Git 仓库。
